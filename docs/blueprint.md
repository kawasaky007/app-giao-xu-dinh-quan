# **App Name**: GiaoXu DinhQuan

## Core Features:

- Authentication: Implement user authentication using Firebase Authentication with email/password.
- Role-Based Access Control: Define and enforce role-based access control (admin, editor, author) using Firestore rules to manage access to different parts of the app.
- Firestore Integration: Utilize Cloud Firestore (Native mode) to store and manage application data including role assignments and user data. The LLM tool determines, according to the role, which resources can be viewed.
- Data Storage: Employ Firebase Storage for storing files such as images or documents.
- Content Display: Display data fetched from Firestore with clear UI.
- Hosting: Deploy the application to Firebase Hosting.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to represent stability and faith.
- Background color: Very light blue (#E8EAF6), almost white, but softer.
- Accent color: Gold (#FFC107) to highlight important actions and information. This is an analogous color chosen for its connection to value and significance.
- Body font: 'PT Sans', sans-serif, is used for clear, readable content.
- Headline font: 'Alegreya', serif, brings an elegant, intellectual touch.
- Employ simple, clean icons that are easily recognizable.
- Maintain a clean and structured layout, making navigation intuitive.
